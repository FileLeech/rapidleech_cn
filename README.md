= 介绍 =

欢迎使用rapidleech，这是一个基于PHP免费开源的网盘及链接转存器。
本版本由Hintay汉化，基于2014年1月12日发布的SVN428。
联系: www.kugeek.com

= 开始使用 =

1. 上传rapidleech到您服务器您想要的目录
2. 浏览器中打开您rapidleech的URL
3. 进行初期设置（请仔细选择）
4. 开始使用吧！
